Please read through the following instructions before start the experiment.

In the experiment, we will conduct experiments to evaluate the two kind of menus: Marking menu and Radial menu.

Trails with different combination of independent variables will be provided and you will be required to select the target object from menu. In this experiment, there are totally 96 trails with 48 each for marking menu and radial menu. For each trail, following information will be given.

Variable table:

  • Trial: Current trail / no. of trials
  • Menu Type: Menu used in the specific trail
  • Menu Depth: Number of layers for the trail, 1, 2, 3
  • Menu Breadth: Number of options in each layer, 2, 4
  • Device: Control device used, Mouse, Touchpad
  • Please select: Target option that you need to select from menu
  • Item Selected: Final option you selected
The using of marking menu and radial menu are provided below.

Marking Menu:
  • Popup: Left Mouse Down
  • Select: Stroke to leaf node
  • Reset: Release Mouse Down
Note: Expert user's can make a fast stroke instead of waiting for the manu to pop up.

Radial Menu:
  • Popup: Right Click
  • Select: Left Click
  • Reset: Right Click
Before real experiment, there is a trail practice section with 8 cases for you to get familiar with the using of these two kinds of menus.
After trail section, you can take few minutes break to relax and prepare. Once you start the real experiment, the following information will be recorded:
  • The starting time and end time for each trail:The starting time and end time will be recorded to calculate the speed.
  • Option selected:The final option you selected will be used to check the acuracy.
  • Attmpt counts:The number of attmpts will be recorded.
If you are ready, please proceed with trail to get familiar with the menus!
